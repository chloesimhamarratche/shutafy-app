# Shutafy Mobile App - Final Project

Shutafy is a mobile-first roommate matching web application.

The goal of the project is to help users find a compatible roommate or offer a room by matching people based on lifestyle, budget, city, cleanliness, religion/practice, smoking, pets and shared values.

Live project:
https://shutafy.vercel.app/

# ⸻

# Project Concept

Finding a roommate is not only about price or location.
People also need compatibility, trust, safety and clear expectations before sharing a home.

Shutafy helps users create a profile, choose their roommate preferences and discover compatible profiles through a simple swipe experience.

The app is inspired by modern matching apps but adapted to the roommate search experience.

# ⸻

# Backend

The backend is handled with Supabase.

The project uses:

* Supabase Auth for user authentication
* Supabase tables for user profiles
* Supabase tables for roommate criteria
* Supabase tables for roommate profiles and matches

This allows the app to save and retrieve real user data instead of only using static front-end content.

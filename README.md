# Shutafy Mobile App - Final Project

Shutafy is a mobile-first roommate matching application.

https://shutafy.vercel.app/

## Project concept

Find the right roommate. Share more than a home.

The app helps users find compatible roommates based on:
- lifestyle
- budget
- city
- cleanliness
- move-in date
- religion / practice
- smoking and pets
- shared values

## Current MVP screens

1. Profile
2. Criteria
3. Messages
4. Swipe / Match
5. My Shutafim
6. Real Estate Advisors

## Tech stack

- React
- Vite
- CSS
- PWA-ready manifest

## Run locally

```bash
npm install
npm run dev
```
